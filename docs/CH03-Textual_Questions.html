  1. Write a C program to find the sum and average of three given numbers.

    #include 

int main() {
    int num1, num2, num3, sum = 0;
    float average = 0;

    printf("Enter 3 numbers : ");
    scanf("%d %d %d", &num1, &num2, &num3);

  5. Explain scanf() and printf() functions in C.

    printf function is used to print/display values of variables using the standard output device (monitor). It can also be used to print any character of our choice in the monitor. Whereas, scanf function is used to read/input values of variables using the standard input device (keyboard). It is used in a program to take input from the user or where there is necessary for user interaction.

  6. Explain the various character input/output functions in C.

    • getchar() function: It is used to read one character at a time from the standard input device (keyboard).
    • putchar() function: It is used to display one character at a time on the monitor screen.
    • getch() function: It is used to read a character from the keyboard and it does not expect the enter key press.
    • putch() function: It is used to display a character on the monitor.
    • gets() function: It is used to read a string of characters including white space.

  7. Mention the various conversion specifications for data I/O in C.

    For integers:

    • %d: represents a decimal integer value
    • %u: represents an unsigned integer value
    • %o: represents an unsigned octal value
    • %x: represents an unsigned hexadecimal value

    For floating-point numbers:

    • %f: represents a floating point value
    • %e: represents a floating point value in decimal or exponential form

    For characters:

    • %c: represents a single character value
    • %s: represents a string of characters

  9. Write a C program to swap (exchange) the values of two variables A and B without using a temporary variable.

    Hint: The following assignments are used to swap the values:

    1. A = A + B
    2. B = A - B
    3. A = A - B
    #include 

int main() {
    int A, B;

    printf("Enter value of A: ");
    scanf("%d", &A);

    printf("Enter value of B: ");
    scanf("%d", &B);

  10. How can the minimum field width for a data item be specified using the printf() function?

    The minimum field width can be specified using a number before the format specifier in printf(). For example:

    • %5d - specifies a minimum field width of 5 characters for an integer
    • %10f - specifies a minimum field width of 10 characters for a float

    If the value is shorter than the specified width, it will be padded with spaces. If the value is longer, it will be printed in its entirety.

  11. What are the advantages of getche() over scanf()?

    • getche() reads a single character without waiting for Enter key
    • getche() is faster as it doesn't require buffer flushing
    • getche() is more suitable for reading single characters in interactive programs
    • getche() doesn't require format specifiers
